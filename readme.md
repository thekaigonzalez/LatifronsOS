# Snarwin - FreeKSD can suck my butt.

Snarwin is a replacement for FreeKSD with portability, 

## Use

Unix

```

$ ./snarwin-path-setup

./snarwin-path-setup: creating SNARWIN_ENV variable.
./snarwin-path-setup: setting up package.path

$ ./snarwin [args]

```

Windows NT (On Powershell)

```

> .\snarwin-x64.ps1

```
